# 🎯 XGBoard v1.1 - Professional macOS Clipboard Manager

## 📱 What's New

XGBoard is a powerful, native macOS clipboard manager built with SwiftUI that brings Windows-like clipboard functionality to macOS.

## ✨ Features

- **📋 Complete Clipboard History** - Automatically saves everything you copy
- **⌨️ Global Hotkey** - Quick access with `Cmd+F2`
- **🔍 Smart Search** - Find clipboard items instantly
- **❤️ Favorites System** - Mark important items for quick access
- **🖼️ Multi-Format Support** - Text, Images, RTF, and Files
- **✏️ Built-in Editor** - Edit text items directly
- **🌙 Native macOS Design** - Beautiful SwiftUI interface with Dark Mode support

## 🛠 Technical Specifications

- **Platform**: macOS 13.0+ (Ventura, Sonoma, Sequoia)
- **Architecture**: Universal Binary (Apple Silicon + Intel)
- **Framework**: SwiftUI + AppKit
- **Bundle ID**: `com.xgboard.clipboardmanager.v2`
- **Size**: ~450KB (Ultra-lightweight!)

## 📦 Installation Options

### Option 1: DMG Installer (Recommended)
1. Download `XGBoard-v1.1-Installer.dmg`
2. Double-click to mount
3. Drag XGBoard to Applications folder
4. Launch and grant accessibility permissions

### Option 2: ZIP Archive
1. Download `XGBoard-v1.1-macOS.zip`
2. Extract and move to Applications
3. Right-click → Open (first time only)

## ⚠️ Important Notes

- **Accessibility Permission Required**: For global hotkeys to work
- **First Launch**: May request accessibility permissions - this is normal and required
- **Gatekeeper**: App is code-signed for security

## 🚀 Getting Started

1. **Launch XGBoard** from Applications
2. **Grant Permissions** when prompted (accessibility for hotkeys)
3. **Use Cmd+F2** to open clipboard history anytime
4. **Start copying!** Everything will be automatically saved
